Project Node Metadata (XML) Project nodes have XML metadata associated with them. You can gain access to it using the getMetaData and setMetaData methods on the WireTapNodeHandle object for the project node. Valid Values for Project Metadata The following table shows valid values for each element in a project XML stream. Some elements are read-only. Values are returned by the IFFFS Wiretap server when you call getMetaData on the WireTapNodeHandle object for the project node.
